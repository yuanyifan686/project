# AI智能体课程咨询网站 产品需求文档（PRD）

版本：V1.0  
项目名称：AI智能体课程咨询平台  
产品类型：AI教育课程展示 + 智能客服咨询网站  
技术方案：Vue3 + Python + SQLite + Coze AI Agent

---

# 1. 产品概述

## 1.1 产品背景

随着AI大模型、智能体（AI Agent）技术快速发展，越来越多企业和个人希望学习AI应用开发。

本项目旨在打造一个AI智能体课程咨询网站，通过课程展示、教师介绍、智能客服咨询等功能，实现用户快速了解课程并完成咨询转化。

网站内置AI智能客服能力，为用户提供：

- 课程咨询
- 学习路线推荐
- 课程价格介绍
- 开课时间查询
- 常见问题解答

智能客服由Coze Agent提供能力，网站预留接口，后续可直接接入。
